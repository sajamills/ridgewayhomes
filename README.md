# Ridgeway Homes — Website Template

Next.js site for Ridgeway Homes / John W. Ridgeway Contractors, LLC.
Deploy to Vercel in minutes.

## Deploy
```bash
npm install -g vercel
cd ridgewayhomes
npm install
vercel
```

## Swap in before pitching
